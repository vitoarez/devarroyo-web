Política de cookies

La presente Política de Cookies explica el uso de cookies, almacenamiento local y tecnologías similares en el sitio web DeVArroyo, accesible desde https://devarroyo.com y https://www.devarroyo.com.

1. ¿Qué son las cookies?

Las cookies son pequeños archivos que se almacenan en el dispositivo del usuario cuando visita una página web. Pueden utilizarse para recordar preferencias, facilitar el funcionamiento de una web, analizar el uso del sitio o prestar determinados servicios.

Además de cookies, una web puede utilizar tecnologías similares, como el almacenamiento local del navegador, también conocido como localStorage, para guardar determinadas preferencias técnicas.

2. Tecnologías utilizadas en este sitio web

Este sitio web utiliza las siguientes tecnologías:

  • Almacenamiento técnico local: se utiliza para recordar la preferencia de idioma del usuario y la decisión sobre aceptación o rechazo de la analítica.
  • Google Analytics 4: se utiliza únicamente si el usuario acepta expresamente la analítica mediante el banner de cookies.

Si el usuario rechaza la analítica, Google Analytics 4 no se carga desde este sitio web.

3. Cookies técnicas o necesarias

Las cookies técnicas o tecnologías equivalentes son aquellas necesarias para el funcionamiento básico del sitio web o para recordar preferencias solicitadas por el propio usuario.

En este sitio web se utiliza almacenamiento local para:

  • Recordar el idioma elegido por el usuario.
  • Recordar si el usuario ha aceptado o rechazado la analítica.

Estas tecnologías no se utilizan para elaborar perfiles comerciales ni para mostrar publicidad personalizada.

4. Cookies analíticas

Las cookies analíticas permiten obtener información estadística sobre el uso del sitio web, como número de visitas, páginas consultadas, tipo de dispositivo, navegador utilizado o interacción general con los contenidos.

Este sitio web puede utilizar Google Analytics 4, un servicio de analítica proporcionado por Google, únicamente cuando el usuario haya aceptado expresamente su uso.

La información obtenida se utiliza para entender cómo se usa la web y mejorar su contenido, estructura, usabilidad y rendimiento.

5. Finalidad de Google Analytics 4

Google Analytics 4 se utiliza con las siguientes finalidades:

  • Medir visitas al sitio web.
  • Conocer qué secciones reciben más interés.
  • Analizar la interacción general de los usuarios con la web.
  • Detectar posibles mejoras de contenido, navegación y experiencia de usuario.

Este sitio web no utiliza Google Analytics 4 para mostrar publicidad personalizada, crear perfiles comerciales individuales ni vender datos personales a terceros.

6. Proveedor de las cookies analíticas

El proveedor de la herramienta de analítica es Google.

  • Servicio: Google Analytics 4
  • Proveedor: Google Ireland Limited / Google LLC
  • Finalidad: analítica web y medición estadística
  • Activación: solo tras aceptación expresa del usuario

7. Aceptación o rechazo de cookies

Al acceder por primera vez al sitio web, el usuario verá un banner de cookies que permite:

  • Aceptar analítica: activa Google Analytics 4.
  • Rechazar analítica: impide que Google Analytics 4 se cargue desde este sitio web.

La decisión del usuario se guarda en el navegador para evitar mostrar el aviso en cada visita.

8. Cómo cambiar la configuración

El usuario puede cambiar su decisión eliminando los datos de navegación del sitio web desde la configuración de su navegador. Al volver a visitar la web, se mostrará de nuevo el banner de cookies.

También puede bloquear, eliminar o configurar las cookies desde las opciones del navegador que utilice habitualmente.

9. Cómo desactivar cookies desde el navegador

Cada navegador permite gestionar cookies y almacenamiento local desde su propia sección de configuración. Normalmente se encuentra dentro de apartados como “Privacidad”, “Seguridad”, “Cookies”, “Datos del sitio” o “Historial”.

Si el usuario bloquea determinadas cookies o tecnologías de almacenamiento, algunas preferencias de la web, como el idioma seleccionado o la decisión sobre analítica, podrían no conservarse correctamente.

10. Transferencias internacionales

El uso de servicios de Google puede implicar tratamiento de información mediante infraestructuras ubicadas fuera del Espacio Económico Europeo.

Dichos tratamientos se realizarán conforme a las garantías previstas por la normativa aplicable en materia de protección de datos.

11. Actualizaciones de la Política de Cookies

Esta Política de Cookies puede modificarse cuando se produzcan cambios técnicos, legales o funcionales en el sitio web.

En caso de cambios relevantes, se actualizará la fecha indicada al final de esta página.
