Política de privacidad

La presente Política de Privacidad explica cómo se tratan los datos personales que puedan facilitar los usuarios al acceder o contactar a través del sitio web DeVArroyo, accesible desde https://devarroyo.com y https://www.devarroyo.com.

1. Responsable del tratamiento

El responsable del tratamiento de los datos personales es:

  • Titular: Víctor Arroyo
  • Marca personal profesional: Víctor Arroyo
  • Nombre del sitio web / identidad digital: DeVArroyo
  • País: España
  • Correo electrónico: contacto@devarroyo.com

2. Finalidades del tratamiento

Los datos personales que el usuario pueda facilitar serán tratados con las siguientes finalidades:

  • Atender consultas, solicitudes, propuestas profesionales o comunicaciones enviadas por correo electrónico.
  • Gestionar posibles contactos relacionados con oportunidades profesionales, colaboraciones, proyectos o procesos de selección.
  • Mantener la seguridad, disponibilidad y correcto funcionamiento técnico del sitio web.
  • Analizar de forma agregada el uso del sitio web mediante Google Analytics 4, únicamente cuando el usuario haya aceptado expresamente la analítica.

3. Datos tratados

En función del uso que el usuario haga del sitio web, podrán tratarse las siguientes categorías de datos:

  • Datos identificativos y de contacto, como nombre, apellidos, dirección de correo electrónico u otros datos incluidos voluntariamente en una comunicación.
  • Datos profesionales que el usuario pueda incluir en mensajes, propuestas o comunicaciones.
  • Datos técnicos básicos derivados de la navegación, como información del navegador, dispositivo, páginas visitadas o interacciones con el sitio web, cuando el usuario acepte la analítica.

Este sitio web no solicita categorías especiales de datos personales. Se recomienda no enviar información sensible, confidencial o innecesaria a través del correo electrónico.

4. Base jurídica del tratamiento

El tratamiento de los datos personales se basa en las siguientes bases jurídicas:

  • Consentimiento del usuario: cuando el usuario contacta voluntariamente por correo electrónico o acepta el uso de analítica.
  • Interés legítimo: para mantener la seguridad del sitio web, responder comunicaciones profesionales y conservar trazabilidad básica de los contactos.
  • Aplicación de medidas precontractuales: cuando la comunicación esté relacionada con una posible colaboración, contratación, proyecto u oportunidad profesional solicitada por el usuario.

5. Conservación de los datos

Los datos personales se conservarán únicamente durante el tiempo necesario para atender la finalidad para la que fueron facilitados.

Las comunicaciones recibidas por correo electrónico podrán conservarse mientras exista una relación profesional, una oportunidad en curso o mientras puedan derivarse responsabilidades legales.

Los datos asociados a la analítica web se conservarán conforme a los periodos configurados en Google Analytics 4 y a las condiciones establecidas por Google.

6. Comunicación de datos a terceros

Con carácter general, no se venderán, alquilarán ni cederán datos personales a terceros.

No obstante, determinados proveedores técnicos pueden tener acceso a datos estrictamente necesarios para prestar sus servicios, por ejemplo:

  • Proveedores de alojamiento web, publicación o infraestructura técnica.
  • Proveedores de correo electrónico y gestión del dominio.
  • Google Analytics 4, únicamente si el usuario acepta expresamente la analítica.

Estos proveedores actuarán conforme a sus propias condiciones, políticas de privacidad y, cuando corresponda, acuerdos de tratamiento de datos.

7. Transferencias internacionales

Algunos servicios técnicos utilizados para el funcionamiento, publicación, correo o analítica del sitio web pueden estar prestados por proveedores ubicados fuera del Espacio Económico Europeo o que utilicen infraestructuras internacionales.

En esos casos, dichas transferencias se realizarán conforme a las garantías previstas por la normativa aplicable en materia de protección de datos.

8. Google Analytics 4

Este sitio web puede utilizar Google Analytics 4 para obtener información estadística sobre visitas, páginas consultadas e interacción general con la web.

Google Analytics 4 solo se carga si el usuario acepta expresamente la analítica mediante el banner de cookies. Si el usuario rechaza la analítica, la etiqueta de Google Analytics no se cargará desde este sitio web.

La información obtenida mediante Google Analytics se utiliza con fines estadísticos y de mejora del sitio web, no para elaborar perfiles comerciales individuales ni para mostrar publicidad personalizada desde este sitio.

9. Derechos de los usuarios

El usuario puede ejercer sus derechos de acceso, rectificación, supresión, oposición, limitación del tratamiento, portabilidad y retirada del consentimiento, cuando resulten aplicables.

Para ejercer estos derechos, el usuario puede enviar una solicitud al correo electrónico:

contacto@devarroyo.com

La solicitud deberá indicar claramente el derecho que se desea ejercer y permitir identificar razonablemente al solicitante.

Asimismo, el usuario tiene derecho a presentar una reclamación ante la Agencia Española de Protección de Datos si considera que el tratamiento de sus datos personales no se ajusta a la normativa aplicable.

10. Seguridad

El titular aplicará medidas técnicas y organizativas razonables para proteger los datos personales frente a accesos no autorizados, pérdida, alteración o uso indebido.

No obstante, el usuario debe tener en cuenta que ningún sistema de transmisión de datos por Internet puede garantizar una seguridad absoluta.

11. Menores de edad

Este sitio web no está dirigido específicamente a menores de edad ni solicita deliberadamente datos personales de menores.

Si un menor desea contactar a través del sitio web, deberá hacerlo con autorización de sus padres, tutores o representantes legales.

12. Modificaciones de la Política de Privacidad

El titular se reserva el derecho a modificar la presente Política de Privacidad para adaptarla a cambios legales, técnicos o de funcionamiento del sitio web.

En caso de cambios relevantes, se actualizará la fecha de revisión indicada al final de esta página.

13. Contacto

Para cualquier cuestión relacionada con esta Política de Privacidad, el usuario puede contactar mediante el correo:

contacto@devarroyo.com
